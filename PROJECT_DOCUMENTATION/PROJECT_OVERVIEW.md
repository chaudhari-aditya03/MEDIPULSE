# Project Overview

## Project Name: Doctors Appointment System

### Description
A comprehensive MERN stack application for managing doctor appointments, hospital resources, ambulance services, blood bank, and more. The system includes multiple user roles (admin, doctor, patient, ambulance driver, donor, etc.) and real-time features.

### Main Modules
- Authentication & Authorization
- Doctor Management
- Patient Management
- Appointment Scheduling
- Ambulance Management
- Blood Bank & Donor Management
- Hospital Management
- Emergency Services
- Visitor Counter & Analytics

### Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Other:** Socket.IO (real-time), JWT, REST APIs
